## 使用方式

- 既有教材檔視為來源材料，不主動重新命名、刪除或覆蓋。
- 新產出的檔案放在清楚命名的課程資料夾或 `outputs/` 子資料夾。
- 保留中文檔名與課程標題。
- 大型媒體與 Office 檔案預設不納入 Git，除非明確要求追蹤。
